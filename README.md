# FitZone Hub - Gym Website

## Overview
FitZone Hub is a simple gym website built using **Flask**.  
It shows Home, About Us, and Membership pages with a responsive layout.

## Features
- Home page with welcome message  
- About Us page with gym info  
- Membership page with plans  
- Easy navigation with navbar  

## How to Run Locally
1. Clone the repo:
```bash
git clone https://github.com/yourusername/FitZone-Hub.git

2.Go to project folder:

cd FitZone-Hub
Install dependencies:

pip install -r requirements.txt


Run the app:

python app.py


Open in browser: http://127.0.0.1:5000/
